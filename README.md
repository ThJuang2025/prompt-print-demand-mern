# AI E-Commerce Wireframe System (Redesigned)

This project contains a complete, redesigned set of low-fidelity wireframes for an AI-powered E-Commerce website.
The system focuses on a modern, clean, and minimal aesthetic using grayscale shapes and generous spacing.
Built with HTML and Tailwind CSS (via CDN).

## Screens Included

1.  **Landing Page** (`index.html`)
    *   Modern header with centered search.
    *   Clean product grid with "Customize" call-to-action.
2.  **Product Customization** (`customization.html`)
    *   **Unified Interface**: Includes AI Design Generator and Product Options.
    *   Large interactive mockup preview.
    *   Tabbed controls for Design vs Product.
4.  **Shopping Cart** (`cart.html`)
    *   Modern list view for items.
    *   Sticky order summary card.
5.  **Checkout** (`checkout.html`)
    *   Clean shipping form.
    *   Order summary sidebar.
6.  **Order Confirmation** (`confirmation.html`)
    *   Centered success card.
    *   Horizontal delivery timeline.
7.  **Authentication**
    *   **Login** (`login.html`): Centered card layout.
    *   **Register** (`register.html`): Account creation form.
    *   **Forgot Password** (`forgot-password.html`): Reset flow.
8.  **User Dashboard** (`user-dashboard.html`)
    *   Tabbed interface (Profile, Orders, Dashboard).
    *   Activity chart and order history.
9.  **Admin Dashboard** (`admin-dashboard.html`)
    *   Sidebar navigation.
    *   Revenue/Sales charts and inventory table.

## How to View

Open `index.html` in your web browser.
